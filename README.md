# IntelliBuy

## Env
  * Java SE 1.8.0
  * Maven
  * Tomcat 9.0
  * Eclipse JAVA EE IDE Photon or higher

## Feature
    Hello Intellibuy.
    Homepage: localhost:8080/IntelliBuy
    
    Show products. 
    Add product to cart. Save in cookie.
    Checkout to create order.
    Login.
    Register.
    Save login state in cookie.
    Add user role in session.
